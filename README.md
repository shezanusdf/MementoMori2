# InspoGrid - Visualize Your Life in Weeks

InspoGrid is a "Memento Mori" wallpaper generator that visualizes your life in weeks. Create beautiful iPhone wallpapers showing a grid where each dot represents one week of your life.

## Features

- **Life Visualization**: Grid of ~4000 dots (52 weeks x life expectancy years)
- **Personalization**: Birth date input, country-based life expectancy
- **Theme Selection**: Light, Dark, Midnight, Sepia, Ocean, Forest, Custom
- **Device Support**: All modern iPhones from SE to iPhone 17 series
- **Appearance Options**: Dot shapes (circle, rounded, square), widget position adjustments, labels toggle
- **Download**: Generate and download high-resolution wallpaper PNG
- **iOS Shortcut**: Auto-update wallpaper daily via iOS Shortcuts automation

## Design

- Warm, Wabi-inspired aesthetic with Instrument Serif + Inter fonts
- Terracotta/coral primary accent color
- Clean, editorial, minimalist layout
- Smooth fade-in animations and floating iPhone mockup

## Tech Stack

- **Frontend**: React + Vite + TypeScript + Tailwind + shadcn/ui
- **Backend**: Hono + Bun

## How It Works

1. Enter your birth date
2. Select your country (sets life expectancy)
3. Choose a theme and customize appearance
4. Preview in the iPhone mockup
5. Download your personalized wallpaper
